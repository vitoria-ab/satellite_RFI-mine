# satellite_RFI
2021/05/04 - Satellite RFI code for MeerKLASS

The code setuo is broken down into stages:

Stage 1:
Calibration of the MeerKLASS data

Stage 2:
Calculating the positioning of the satellites with respect to the MeerKAT pointing

Stage 3:
Simulation construction of satellites and fitting to the Observational data
